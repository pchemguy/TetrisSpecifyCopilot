# Classic Browser Tetris

This repository contains a React + TypeScript implementation of Classic Browser Tetris.

## User Documentation Entry Point

Start here for install, launch, controls, scoring, persistence behavior, and troubleshooting:

- [User Guide](docs/user-guide.md)

## Quick Start

Windows users require Git Bash (for example, Git for Windows) or WSL; PowerShell is not supported.

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite in your browser.
